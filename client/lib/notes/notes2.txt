----->  Why signup page using stateful widget?
---------------------------------------------------------------------------------
 --> A SIGNUP SCREEN isn’t just static text; it needs to remember what the user has
typed and update the UI as things happen. That’s why it’s built as a StatefulWidget.

--> you can think of the State object as the place that
 stores and keeps track of the changing state of that widget.
